Principais Formatações

Negrito ou Destaque

Nesta frase temos um termo em negrito usando a tag <B> (não semântica)

Já nesta frase, tenho um termo de destaque usando a tag <STRONG> (semântico)

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <i> (não semântico)

Já nesta frase, temos um termo em ênfase usando a tag <em> (semântico)

Texto marcado

Podemos criar também um texto marcado com a tag <mark>.

Ou também marcado com CSS para mudar a cor do mark.

Texto pequeno

Podemos criar um texto pequeno com a tag <small>.

Texto deletado

Podemos criar um texto deletado, isto é, lido, mas não considerado por meio da tag <del>

Texto inserido

Pode-se marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado depois com a tag <ins>.

Texto sobrescrito

Para inserir coisas elevadas à potência como X6+2. A tag usada é <sup>.

Texto subscrito

Para inserir coisas do tipo H2O. A tag é <sub>.
